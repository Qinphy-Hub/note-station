# 格特露德

> [译] 胡鼎其

&emsp;&emsp;黑塞好多书都没办法讲清楚写的是个什么主题。但这本书还是要比《克林索尔的最后夏天》和《彼得·卡门青德》更具故事感，共鸣也稍多一些。其实，主题不主题不重要，一千个读者就有一千个哈姆雷特，不同的人总能从这本书中读到不同的内容。

&emsp;&emsp;我想我读到了内容是，关于知己。主人公喜欢音乐，一步步找到适合自己的事。但一路上都是充满打击，不被认可，甚至失去了活着的意义。直到遇到了摩特，一个能欣赏他才华的人。我想很多人或许不能理解主人公与摩特之间的友谊，明明没发生太多的事，为什么就能达成密切的关系。我认为，在长期的父母、老师、同学都不认可自己的环境下，一个能认可自己的能力的人，那一刻就已经达成了朋友关系。但是摩特也仅仅是对主人公的认可，他缺乏对主人公的理解，而格特露德达到了。这就是主人公最想要的知己。但可惜，主人公倾心于她，她却喜欢摩特。当然，很多人也无法理解主人公不主动的性格，为何不去争取呢？特别是在摩特死后。我认为主人公母亲和她的朋友就是一个例子。有时候，我们会认为一件事不去做就会留下遗憾，但有时候做了亦会成为伤痕。其实，人生得一知己足矣。她安好地在那儿，足以。这是支持主人公活下去、面对生活、面向未来的一切力量。
