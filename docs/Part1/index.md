# 阅读札记

!!! abstract "2026阅读进度"

    * 2025某天 - 2026.01：[《朋友》黑塞](./Hesse/freunde.md)
    * 2026.01.01 - 2026.02.08：重温[《月亮与六便士》毛姆](./moon.md)

&emsp;&emsp;高中毕业时，感觉自己终于自由了，开始写人生计划。进入大城市，令人眼花缭乱，想法太多，早就把当初的计划抛于脑后。有一天，突然感觉不能再这么追名逐利下去了。我又重新拿起笔，开始按自己内心写计划。我记得我想要认识整个世界，我想要读书，先看看别人眼中的世界。于是，在2019年初，我开始了阅读计划。阅读的第一本书就是《月亮与六便士》。因为，我也想知道我一直追寻的理想为什么就不如六便士了？但可惜当时还是年轻，没有看懂。

&emsp;&emsp;自从开启阅读计划之后，我开始写一些读书札记。但可惜，写计划时总是热情满满，执行起来热情逐渐消退，效率越来越低。2019年看了18本，2020年看了14本，2021年看了8本，2022年和2023年各看了一本。一开始喜欢看历史类书籍，觉得以史为鉴可以正衣冠。后来也看传记，但觉得没什么趣味性。后来看书越来越少，是因为考上研究生之后，就进入了一个迷茫的阶段。

&emsp;&emsp;我知道我从小就有想从事物理相关研究的执念，只可惜，有这个心没这个本事。没有走上这条路，但这个执念一直在暗处指引我，要我走上那条路。本来在我那几年，计算机就业形势可好了，我也找到一个不错的工作，但我考上研究生就放弃了工作。或许在现在看来，读研就是比较好的选择，但在当时就业形式过好，考研反而耽误升职发展。毕竟程序员有一个35岁大坎。在2023年到2024年之际，一本《悉达多》突然让我茅塞顿开。无非是顺着自己的意志走下去，那就走吧，不再担心未来会发生什么。

&emsp;&emsp;可能是多年来读论文增强了我的阅读理解能力，我写的读书札记越来越长。我从站在道德、逻辑的角度批判虚拟人物，到能够站在人物的角度思考，再到现在能够结合自身处境进行反思；我是真的觉得，自己在阅读理解上有了很大的进步。想到当初，初高中的时候，语文老师们都无法理解，作为一个中国人，语文考试能够不及格。{==所以，还是不要期待我的札记里面，能有什么精辟之语了==}。
