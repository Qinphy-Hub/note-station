# 朋友

> 王滨滨译本

&emsp;&emsp;读完感觉温馨中带点伤感，伤感的是汉斯与埃尔温之间留下了遗憾，但让人感动的是他们最终都真正看见了彼此，一切尽在不言中。

&emsp;&emsp;埃尔温首先在这段关系中看见了自己。在矛盾中，埃尔温成长了，独立了，学会了不再围着汉斯转：

!!! quote
    尽管这位令他钦佩的朋友充满才智，但毕竟不是个英雄。

&emsp;&emsp;汉斯在这段关系中学会看见对方，看见埃尔温。

!!! quote
    他惊愕地发现，自己在长达多年的友谊中完全不像他一直以为的那样洞察一切、知晓一切，而是相反，埃尔温比其他任何人都了解他，但他几乎不了解埃尔温。

&emsp;&emsp;不得不说，汉斯与我有太多相似之处。当我面对相同的情景时，我会做出类似的决策；而他遇到的问题，我也曾遇到。我也在关系中看不到对方；爆发矛盾时，看似专注自己的选择，实际上是一种逃避；而真是要有维尔特这样强大的人，我也会毫无抗拒之力地被吸引，就算是纸上的人物，都已引起了我的兴趣。不过，最后和汉斯的感受也是一样的，发现除了精神世界以外，还有更值得珍惜的东西。

&emsp;&emsp;看到一条豆瓣读书的书评：“一个自以为是的INTJ”。这让我哭笑不得，虽然不愿意被贴上INTJ的标签，可处处都命中INTJ的特征。让我有一种被人看穿的危机感。
