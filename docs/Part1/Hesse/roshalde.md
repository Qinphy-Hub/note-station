# 罗斯哈尔德

> 莫光华 译

&emsp;&emsp;看这本书之前，听网上说这本书的主题是“婚姻”，这还让我蛮感兴趣的，让我为即将到来的事情做好准备。看完之后，不，这个总结太过狭隘。我认为书的主题应该更大一些，是家庭，是家庭中个体的精神困境。

&emsp;&emsp;个体精神的困境可能有很多人无法理解，就像我在评论区看到这样的评论：艺术家就不该结婚。非也！就算是其他人也有着自己的追求和热爱，这就是他们的精神。但是在家庭中，这些精神总是要让步于家庭的需要，毕竟两个情投意合且志趣相投的人可太难相遇了。就算如此，也无法保证他们的孩子能够与他们有一样的精神追求。亦或许是我还没有步入婚姻，也对他们的精神困境到如此不可调和的地步没有多少理解。也许是性格原因，我喜欢寻求两全之策，经验之谈，现实中在付出能够接受的牺牲下，我还是总能找到的。但这不重要，这不是这本书的中心，就像《月亮与六便士》史特利克兰德离家出走的决定有没有两全之策那样不重要。他们的关系就是不可调和了，就像现实中那些想要离婚的人一样，有我们无法理解的原因。

&emsp;&emsp;但最可怜的是他们孩子小皮埃尔的精神困境。他生活在这样一个父母不相爱的环境中，对他也是如此。因为沉迷于自己的精神世界中，约翰对爱他的孩子忽视得太多。他总是关心皮埃尔以后想当什么，却忽视皮埃尔皮埃尔当下的需求。我真是被这种被忽视、不被理解的困境给抓住了，我特别能理解皮埃尔的痛苦。甚至与皮埃尔做过相似的梦，无论怎么呼喊都无人应答，哪怕他们近在咫尺。这种感觉真是悲伤、绝望、痛苦和孤独。

&emsp;&emsp;再次感叹一下黑塞的书，能写出这么多与我有相似感觉、相似行为的细节。这要么是观察细致入微，要么是亲身经历。但我想他的大部分作品中的人物恐怕都是他自己的映射吧！
