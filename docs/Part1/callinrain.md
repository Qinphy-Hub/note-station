# 在细雨中呼喊

> 作者：余华

&emsp;&emsp;我记得我第一次下定决心要开始阅读时，读的就是余华的《活着》，当时把我感动得一把鼻涕一把泪。这么多年过去了，阅读计划断断续续了很多次，我肯定这一次的决心是真实的！读这本书出于某种巧合。之前很长一段时间，都在听App随机推荐的一些歌曲，每一次李健的歌都给我比较深的印象。继陈奕迅以来，我认为我也喜欢李健的风格的。当我在微信读书中刷到这本书，介绍的第一句话就是：李健认为这是余华最好的作品。于是我毫不犹豫地就开启了阅读。

&emsp;&emsp;从一个孩子的眼中看这个世界，非常细腻的描写，让人特别容易共情。他的一路成长，对家的认识、对友情的理解、对自己的感知、对异性的想象，被事件冲击、被真相刷新、被时间重塑、被温情修补，就如那次南门的火焰一样，在细雨中呼喊。

&emsp;&emsp;主人公成长中的每一个故事都很精彩，但让我共情深刻的是他在家庭中的艰难生存，以及他在友谊中的成长。不知道现在有多少人在家庭中有过被忽视的感觉，甚至后知后觉。就我周围和网络上来看，这样的家庭应该不在少数。正如苏宇的感受那样，他既感激母亲的照看，又伤感于不被理解和关心。更别说孙光林这般，被排挤和嫌弃的。两个相似的人，最终吸引到一起，抱团取暖。他们共同成长的友情，教会了彼此缺失的“爱”。在苏宇走后，孙光林把这样的“爱”传递给了同样遭遇的鲁鲁。这就是让读者感动的成长，源于友谊的相扶相持，然后薪火相传。

&emsp;&emsp;孙光林对自己的认识、对异性的想象也让我刷新认知。一直以来，我都很好奇，对异性的“爱”到底是什么。在孙光林的成长中，似乎很好地区别了“爱欲”和“性欲”。原来，男生在成熟以前，也会受到女生的吸引，这是一种本能的对美的欣赏，是一种“爱欲”；而在性成熟以后，“性欲”就会参与进来。两种欲望的叠加，使其行为更加难以克制。而这更能区分友谊之“爱”，友谊的“爱”更需要一种相似和相知。当然，我认为大家心目中的“爱情”更像是这三种“爱”的叠加。

&emsp;&emsp;还有很多重要的故事，比如，孙光林的父亲，就是纯恶。我读的时候始终都在找这样性格的人是怎么形成的，然而就是没有原因，然而确认他就是一个纯粹的大坏蛋。还有一些有趣的孩童之间的小故事，比如，“你骂鬼神的时候，在害怕”，等这类孩童真言。总之，这里的故事值得细细体会。
