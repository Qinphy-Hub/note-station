# 彼得·卡门青德(又名：乡愁)

> 译者：胡其鼎

&emsp;&emsp;黑塞的成名作，文笔还有些稚嫩，故事的起伏也还有些生硬，让人看了摸不清主题。有时候阅读要什么主题呢？不过是跟着作者的步伐，看作者之所看，感作者之所感而已。

&emsp;&emsp;黑塞的大部分作品，都是在讲述一场修行，而《彼得·卡门青德》就是黑塞修行的起点。修行的开始源于信仰的力量，有信仰才有修行，有修行才会明白信仰的本质。

&emsp;&emsp;彼得自认为是“山里人”，本质上“山里人”只是每一个不被发现、不被理解的内心，要了解它就得翻山越岭。所以，彼得一开始就是孤独的，直到遇到一个能理解他内心的朋友。那种瞬间爆发的表达欲，是我当前能够特别感同身受的。但这只是修行的开始，当自己的内心不断被人发现，但又不断被世界摧毁，在一系列苦难面前，彼得慢慢闭上的他那“聒噪”的嘴，在孤独中，他开始学会聆听自己内心的声音。
