# 荒原狼

> 姜乙译本

&emsp;&emsp;主人公荒原狼哈里·哈勒并不是一个讨喜的角色，他性情孤僻与世人格格不入。他自己也清楚知道这一点，并深感孤独。他称自己身体里有一半是人，有一半是狼，并且他们相处并不和谐。作为人的一半会嫌弃狼的原始冲动，作为狼的一半总是嘲笑人的虚伪，这让哈里痛苦不堪。书中有一则《论荒原狼》的小册子并不赞同荒原狼把自己性格一分为二的做法，人性格是多元的，没有是非对错，这也是这本书想要表达的观点。

&emsp;&emsp;或许每一个初出茅庐的年轻人都会经历这么一个相当挣扎的阶段吧。年轻人的心里装着是非，眼里只有黑白，甚至这种被审视的对象也包括他自己。我想我也是刚刚经过这个阶段吧，对“人人都是与众不同的”这句话有了更深刻的理解。但有点可惜的是，那些我曾厌恶的本性早已被我戒除掉了。然后发现自己变得单调，甚至还丧失了一些能力。所以，没有什么绝对的黑白，就让该存在的东西存在吧，好的坏的都是自己的一部分。

&emsp;&emsp;作者黑塞也给了这类挣扎中的年轻人一个回答，这是我的总结：“毁灭从前，经历现在，重塑未来”。我认为《荒原狼》与《悉达多》是同一种人的两个故事版本。哈里有自己的标准，悉达多也有自己的追求。不同的是，悉达多有盲目追求理想的条件，他是婆罗门的王子，走到哪儿都是人们尊重的对象。而哈里只是一个普通人，他追逐自己的内心就注定孤独。可无论是哪个故事，作者都告诉读者，内心的答案不在内心，在世间。我有这样一个理解：人不可能生来就独立，你所坚持的是非黑白都是别人树立的，一个世俗意义上的完人。而我们实际上追寻的是真实的自己，眼里却只看得见由条条框框约束出来的雕塑，怎能不痛苦挣扎。所以，放下那个光辉形象，毁了那些条条框框，去体验世间本身存在的一切，去构建一个多元的自己。这就是悉达多所追寻的真理，人本身，也是荒原狼的救赎。

&emsp;&emsp;就像最后在帕布罗的魔术剧院中，最后一幕，哈里杀死了躺在帕布罗身旁的赫米娜。帕布罗并不完全满意，因为哈里的行为不是出于嫉妒，而是出于对同类的理解，想要求死的孤独之人。但魔术已经很成功了，哈里踏出了第一步，接受了身体里的狼性；而这场刺杀也预示着哈里杀死了过去的自己，重新开始人狼共处。
