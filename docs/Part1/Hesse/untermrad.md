# 在轮下

> 姜乙译本

&emsp;&emsp;居然是一个悲剧，汉斯还是被碾压在车轮之下了。这是一个相较于黑塞其它作品更加简单的一个故事，却让我感到万分悲恸。

&emsp;&emsp;黑塞笔下的人物，让我的代入感都很强。但是对于《在轮下》里的故事，我相信有很多中国的学生都会感同身受。向前、向前，朝着唯一的目标不停向前，让假期失去了意义，学习失去了真谛。上大学时以为自己跟随的是时代的巨轮，上着最热门的专业——计算机，毕业时互联网寒潮还没来。硕士阶段做新能源，那时候光伏拿着巨额补贴蓬勃发展，毕业时还没到内卷的时候。博士阶段做“车-路-网”，新能源汽车带来的广泛前景，又不知道什么时候会暴雷。我看似一直站在时代的潮流上，何尝不是被社会运转之轮追赶呢？当读到汉斯不明不白地漂浮在河水中时，惋惜一条生命被碾压，叹息自己还在挣扎。

&emsp;&emsp;这个时代有多少汉斯这样的人？真正的悲剧不是汉斯的死亡，而是跑在汉斯前面，被埋没、失去精神、甚至也将失去生命的悲剧。尽管如此，黑塞指出了另一条路的存在，像海纳尔那样逃离。这也是黑塞自己的做法。《在轮下》像黑塞的自传，去毛尔布隆神学院上学，之后陷入精神危机而退学，逃学之后追求诗与自然。社会不是缺了谁就不转了，找到自己喜欢的事，或许很普通，但这不重要；让那些看着光鲜亮丽的事，让真正懂得、真正喜欢的人去做吧；我相信这些人都会彼此欣赏的。让那些贪慕虚荣、功利的人，继续被时代裹挟着吧，因为实际上他们没人在乎。
