# 东方之旅

&emsp;&emsp;当我读完第一节的时候，我就知道现在开启这本书还是太草率了。书中有太多经典著作中的经典角色我还没有读过，尽管页脚给了详细的批注。但是看完这本书之后，我觉得是时候了！但是，依然值得我在积累一定的阅读量之后，再来回顾这本书。

&emsp;&emsp;就目前来说，我是这样理解的，东方之旅就是作者的朝圣之路，是作者内心对他的真理的追求。在第一节描述的旅途中，这些陪伴作者的有他喜欢的作品中的人物，也有他的朋友(曾出现在他自己的作品中)。但也有一些人没有走到最后。这样对比的话，我现在的同行者不就是赫尔曼·黑塞吗？

&emsp;&emsp;起初我在都的时候，我还不知道利奥有何象征，他和文件的消失是不是暗示作者的一次信仰的崩塌呢？但在真相揭示之前，我在阅读的间隙就感觉“利奥的消失”似乎象征着作者对“理想的背叛、誓言的逃避”。因为在我的记忆中，我也忘了我的理想是什么。以遗忘为分界线，作者描述的那种孤独和寒意也能够勾起我记忆中的感受。不仅如此，遗忘之后的挣扎和迷惘就与我当前阶段的心态重合。作者想写一本“东方之旅”，这种强烈的表达欲受限于表达能力的限制而备受煎熬。我也想表达，也想将一个故事，却发现也是一团乱麻。无法避开我的“利奥”也是我的烦恼。这是不是说，我其实也想写一本我的“东方之旅”呢？

!!! quote "第三节"

    &emsp;&emsp;就在我做出承诺的瞬间，一阵幸福回忆如阳光般洒落在我身上。因为我想起来了，类似的感受在我心中涌现，就在我们开始远征的时候。

&emsp;&emsp;我现在非常理解这种感觉，就像在读[《悉达多》](./siddhartha.md)的信念时，是一种很幸福很温暖的感觉。我意识到我也曾坚定不移地许下诺言、下定过决心，那种彻底的坚定现在回忆起来也很幸福。我忽然想到，其实我的信仰也已经找回来了，它实际上并不是什么具体的东西，而是内心的一种指引，而现在我要做的，仅仅是跟随它。

&emsp;&emsp;当至高宝座出现时，我立即认为那就是利奥的位置，当作者写道“利奥消失在人群中”的时候，我还以为我理解错了整本书的主旨。没错！利奥就是代表作者的信仰。那场信仰对他的审判也让我思索，如果让我面对那些被我遗忘的誓言，我也会惭愧万分，迫使我找到些借口逃避。一无所有时，诺言总是轻巧的；拥有得太多，就难以放下了。就连这句我以前也是不信的。

&emsp;&emsp;我想这本书给我最大的安慰是，“下莫比奥”那场遗忘信仰的事件只是一场考验。虽然我在这场考验中犯下了很多错误，但我的信仰没有抛弃我。
