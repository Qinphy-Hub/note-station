&emsp;&emsp;阅读对于我来说，并不是一件容易的事情。小学的时候，读课文只是为了识字。在我脑海中，很少有文章让我感兴趣，读完之后什么感觉也没有，像是听了别人家的八卦一样难受，毫无用处又浪费时间。难得几篇触动到我的文章，却在后来让我知道都是虚构的，真是令人受伤。后来还要学习古诗词、文言文，我并不认为“读书百遍，其意自现”这种说法有什么道理，就是读不懂、背不了，到现在也记不得几句。阅读理解更是我无法面对的一座大山，且不说试卷上的文章出处质量如何，我既对答案揣度的作者意图深感怀疑，又对作者表达的含义难以理解。所以，我很抗拒“语文”这门学科，一直以来，成绩都在及格线边缘徘徊。老师们也很无奈，说头一次见学不懂“语文”的。

&emsp;&emsp;我的写作也是一塌糊涂。从小我就缺乏表达的欲望，我不知道写什么故事，更不知道如何才能符合老师要求的结构。但是，有些时候我的文章也能受到语文老师的青睐，成为过“范文”。也就那么一两次高光时刻吧！因为对于那些话题我确实有自己的看法，有一些词藻的积累，加上强烈的表达欲，自然也就会思考如何把我脑子里的想法表达出最好的效果。所以，我还是相信，这些我想要表达的“读书札记”，都把我的想法以良好的效果表达出来了。

&emsp;&emsp;我很想问问大家坚持阅读的理由。对于我来说，阅读计划起源于我想重新认识这个世界，不想再看到课本上的那一套了。所以，最初的阅读列表更倾向于历史、科幻类的读物。一些历史人物传记，还有当时很火的《万历十五年》、《未央宫》、《东周列国争霸记》、《东晋门阀政治》、《中国历代政治得失》这类的。也读过一些稍微枯燥晦涩的，比如《江村经济》、《乡土中国》。我都很喜欢。但是，后来又觉得这些不过是一家之言，还是太主流了，我开始在网上找一些有自己见解的内容。渐渐的，历史类读物就退出了我的读书列表。

&emsp;&emsp;实际上在初高中的时候，我就尝试过读科幻类的作品。当时，看了一些国外人气较高的科幻短篇。但始终无法引起我的兴趣，因为它们最终的落脚点都在老生常谈的那些人性困境，科幻的作用只是给了作者一个便于叙事的工具。不过在阅读计划中，个人认为乌托邦三部曲还是不错的，虽然它们也具有类似的问题。直到刘慈欣的《三体》系列，才让我感受到科幻的真正魅力，但我也不得不吐槽一下，文笔的美感确实不如文学性的小说。

&emsp;&emsp;小说也是阅读列表中的重要角色，但是尝试了很多，都无法都明白。比如《月亮与六便士》、《了不起的盖茨比》，我不知道作者想要表达什么，甚至怀疑故事的合理性。但也有我看了比较有感触的小说，比如《百年孤独》、《白夜行》。但除此之外，很难有一本小说能够吸引我去读完。因为读不到令我印象深刻的作品，我的阅读计划越来越难以坚持下去。直到我的迷茫遇上了黑塞的《悉达多》，尽管一开始我也读得七窍生烟，但经过反复回味之后，突然悟出了一点阅读小说的方法。

&emsp;&emsp;在阅读小说时，我常常把自己带入到主人公的视角中去，这并没有什么问题。但问题在于我不仅把自己带入进去，还顺手带上了道义、逻辑等批判的武器。于是很多时候，我都是在吐槽，谁谁谁行为真是愚蠢，某某某怎么可能存在。再重温[《悉达多》](./Hesse/siddhartha.md)时，悉达多的精神的纯粹、信念的坚定，让我深深陷入自己的回忆。最终，让我放下了评判的标准，真正地顺着作者的描述去体会悉达多的精神世界，才让我茅舍顿开，走出了迷茫。

&emsp;&emsp;我还重温了很久之前也读得义愤填膺的[《月亮与六便士》](./moon.md)，那种感觉就完全不一样了。此时此刻，真是得感叹一下，这么多年，我的阅读总算是入门了。{==我想从黑塞系列开始，记录下我的阅读经历，希望有小伙伴一起交流感受==}。
