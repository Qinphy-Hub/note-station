# 人生海海

> 作者：麦家

&emsp;&emsp;早在去年就看过麦家的《风声》，非常精彩，大概是作者集大成之作。因为其它的作品要么是同一类题材(谍战)，要么看得出都有《风声》的影子。就比如这本《人生海海》，写作结构还是很相似的。给了一个充满秘密的人，引起了一阵阵的“风声”，我们借着主人公的视角，穿过层层迷雾，一步步靠近真相，却从不让主角确认“风声”的一切。纯折磨读者了，让人恨不得一口气读完。这个套路真的太适合讲故事了，我是第一次读到这么会讲故事的作者，值得学习一下这种写作模式。

&emsp;&emsp;但是从阅读感受来说，能写出来的不多。纯粹的故事给人以纯粹的感受，感受上校命运的浮沉，感受世间人欲望的碰撞。“人生海海”原是闽南话形容人生的宽广、拥有一切可能性。但是从上校的故事中，我感受到再宽广的人世间，一个人膨胀的欲望只能被其他人膨胀的欲望所制止。像上校这么纯粹的人，只能被别人挤压，艰难地存活着。
