# 线性规划

&emsp;&emsp;线性规划是学习运筹学的入门课题：

* 需要有线性代数的基础，需要的知识也不是特别深入：解方程、矩阵及其几何意义。
* 此笔记是根据《Introduction to Linear Programming》编写的，作者为Dimitris Bertsimas & John N. Tsitsiklis。
* 此笔记主要涉及线性规划部分，拓展内容比如整数规划部分，不会详细介绍。
