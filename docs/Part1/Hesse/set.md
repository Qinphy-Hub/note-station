# 黑塞其它文集

!!! quote "图书列表"

    * 《克林索尔的最后夏天》张佩芬 译
    * 《我独自一人，却很自在》张佩芬、钱春绮、韩耀成 译
    * 《我走入宁静蔚蓝的日子》窦维仪 译

&emsp;&emsp;《克林索尔的最后夏天》实际上是饱受推荐的，但是在读了黑塞这么多作品之后，让我觉得自己可能对他的作品免疫了。当然，更大的可能是这几天很浮躁，看一本书急于求成。在微信读书和语音朗读的加持下，我似乎陷入了一种快餐式阅读。这本书与黑塞其它出名作品风格太不相同了，书中的内容更接近散文而非故事，更加不适合这种阅读方式，反而按照我之前自己逐句朗读的方式更好。在读完《我独自一人，却很自在》之后，我保证，下一本黑塞的作品一定细嚼慢咽。

&emsp;&emsp;《我独自一人，却很自在》、《我走入宁静蔚蓝的日子》这两本书是黑塞的短篇、散文和诗歌集。从作者对自己生活的叙述中，克林索尔的出现频率意外的高。好遗憾没有仔细阅读，我会找时间回顾这本书的。不过当前给我的感觉是，克林索尔的那个夏天就像是一个深受打击的人，向死而生。开始卸下世俗的枷锁，回归自然，感受自然，开始最后的狂欢。让人感觉随心所欲，但这欲望无关世俗的价值，只在于自己的感官。我当前正身处于工作中，对这样的感官体验恰巧是没有耐心的，要是在寒假读这本书就好了。

&emsp;&emsp;在《我独自一人，却很自在》这本小册子中，能把我从恍惚中拉回来的故事是《鸢尾花》一节。我也想感叹大学这几年到底给我灌了什么迷魂汤，让我就像失忆一般。我发现自己很陌生。那些外来的目标、价值使我“神魂颠倒”，它们把我引入歧途。当我在陷阱中困惑、迷茫时，我试图翻找以前的记忆和凭证。真是好奇怪，第一眼会觉得，那是我做过的事吗？这是我写的吗？第二眼又能够感觉的，某种美好而珍贵的东西散发出来。年轻的自己真是惹我喜爱。是啊，太奇怪了，以前我是多么厌恶自己的昨天，期待自己的明天。而现在，我对曾经又有了这么强烈的怀念，对明天已经无所期待了。若如黑塞所说，是时候找回自己的内心了。我知道就如从前走到现在一样，再走回去也一样艰难。

&emsp;&emsp;读黑塞的散文，很宁静，像世外桃源。记得第一次学习散文的时候，是朱自清那篇《荷塘月色》，我觉得枯燥乏味、不知所云。十多年过去了，我才懂得欣赏散文。我记得我曾经有这样的愿望，在远离城市的农村，拥有一个宅子和一片田地，种种花草，看看星空。我记得我的朋友也很赞同我的想法，但是我们都能意识到这太理想了。因为我们不止活在真实的世界里，还活在人类自己构建的虚拟世界——被人称为社会。黑塞的散文再次唤醒了我这段向往田园生活的记忆，让我体验到了我曾经向往的生活。读黑塞的文章就是这一点吸引我，时不时都有那么一两点描写，让人回忆道自己的相似感受。就像找到了同道中人，又像是自己的感受得到了认可。而我最大的收获，就是接受了自己的认定的不好的情绪，无论是嫉妒，还是憎恶，我都可以拥有。于是，发现以前的自己不再那么令人讨厌了。虽然说这道理很简单，无非就是：接受自己的不完美。但很多时候都做不到，只能成为安慰自己的说辞。但读完黑塞的书，会真的让心收获宁静。
