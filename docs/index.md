## 关于网站

!!! note "闲话"
    &emsp;&emsp;遥想当年大学毕业设计，自认为未来自媒体一定会兴起，想要设计一套让没有计算机基础的人也能构建自己的自媒体网站，但能力有限。我设计了在线编辑或上传markdown笔记，上传视频，评论区等，但是还存在的困难是markdown笔记中素材的处理，比如包含视频、图片这类素材，如何配合markown笔记一起上传。另外还有最大的问题是如何教会用户部署服务器，这是最大的问题。
